# deleteSnapshotTemplate


## 描述
删除截图模板

## 请求方式
DELETE

## 请求地址
https://vod.jdcloud-api.com/v1/snapshotTemplates/{templateId}


## 请求参数
|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**templateId**|String|True| |模板ID|


## 返回参数
|名称|类型|描述|
|---|---|---|
|**requestId**|String|请求ID|


## 返回码
|HTTP状态码|错误码|描述|
|---|---|---|
|**200**||OK|

DELETE

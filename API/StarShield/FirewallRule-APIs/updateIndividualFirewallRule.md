# updateIndividualFirewallRule


## 描述
更新一个单独的现有防火墙规则。

## 请求方式
PUT

## 请求地址
https://starshield.jdcloud-api.com/v1/zones/{zone_identifier}/firewall$$rules/{id}

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**zone_identifier**|String|True| | |
|**id**|String|True| | |

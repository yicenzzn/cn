# modifyReplicaNum


## 描述
扩容副本。

## 请求方式
POST

## 请求地址
https://clickhouse.jdcloud-api.com/v1/regions/{regionId}/instances/{instanceId}:modifyReplicaNum

|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**regionId**|String|True| |地域代码|
|**instanceId**|String|True| |实例ID|

## 请求参数
|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**newReplicaNum**|String|True| |修改后的实例副本数|


## 返回参数
|名称| 类型      |描述|
|---|---------|---|
|**result**| Result  | |
|**requestId**| String  | |

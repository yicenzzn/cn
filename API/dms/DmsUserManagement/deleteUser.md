# deleteUser


## 描述
删除用户

## 请求方式
POST

## 请求地址
https://dms.jdcloud-api.com/v1/management:deleteUser


## 请求参数
|名称|类型|是否必需|默认值|描述|
|---|---|---|---|---|
|**dmsUserDeleteList**|String[]|True| |删除用户的pin信息|


## 返回参数
|名称|类型|描述|
|---|---|---|
|**requestId**|String|请求id|


## 返回码
|HTTP状态码|错误码|描述|
|---|---|---|
|**200**||OK|

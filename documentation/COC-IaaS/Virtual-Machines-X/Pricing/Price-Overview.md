# 价格总览

价格总览中价格为统一官方标价，不包括用户折扣优惠，为了保证价格准确性，建议您登录控制台伙伴云主机创建页面查看配置价格。


京东云伙伴主机实例价格主要包括：实例价格（vCPU及内存）/SSD硬盘价格/公网IP带宽/公网IP防御价格；还可基于伙伴云主机创建伙伴快照，用于容灾。

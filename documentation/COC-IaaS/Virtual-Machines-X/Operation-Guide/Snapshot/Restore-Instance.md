# 还原实例

对实例制作快照后，如果实例系统出现异常或错误时，您可以使用伙伴快照还原您的实例，实现实例SSD硬盘数据的版本回退。


## 前提条件及限制

* 已经为您的实例创建伙伴快照，且实例未被删除；
* 伙伴快照处于**可用**状态；
* 要恢复的实例处于**运行中**或**已停止**状态；若实例处于其他非稳定状态，还请等待前序操作执行完成后再操作调整配置。
* 实例仅支持升配，不支持降配，包括实例规格、SSD数据盘、公网IP带宽、防御均仅支持升配；


## 操作影响
* 使用伙伴快照恢复源实例时，实例业务将会中断，且恢复时间较长，请提前做好业务规划。


## 操作步骤
1. 访问[伙伴快照控制台](https://cnsx-console.jdcloud.com/compute/snapshot/list)，即进入实例列表页面。或访问[京东云控制台](https://console.jdcloud.com)点击顶部导航栏**供应链-伙伴云主机**进入伙伴快照列表页。
2. 选择地域。
3. 在实例列表中选择需要还原的伙伴快照。
4. 点击 **操作-还原** 按钮。

![image](https://user-images.githubusercontent.com/88134774/197790730-1aa64702-d78c-4f16-b294-e513b9f414da.png)


4. 在弹出的“恢复实例”弹窗中，确认信息，点击确定提交恢复。

<div align="center"><img src="https://user-images.githubusercontent.com/88134774/197791696-09593911-ba80-4244-9bdc-f5a99771cfcc.png" width="600"></div>

之后伙伴快照将进入“使用中”状态，伙伴快照将无法进行其他操作。当恢复完成后，伙伴快照将恢复**可**状态。

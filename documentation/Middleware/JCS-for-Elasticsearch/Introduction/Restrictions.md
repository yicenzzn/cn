### 网络访问

为了安全起见，云搜索Elasticsearch集群构建在私有网络VPC内，对集群数据的写入和查询也都限定在同一VPC网络内。

```
请注意：
* 云搜索Elasticsearch集群创建成功后，不支持VPC网络的切换，需要您在创建集群时，提前进行业务规划部署。

```

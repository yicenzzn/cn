# 创建子用户

如果您有管理子用户的权限（如您是主账号或 IAM 管理员），可以登录京东云控制台，进入管理 > 访问控制 > 用户管理，点击【创建子用户】，在子用户创建页面定义子用户信息和访问类型。

子用户创建成功后，可以在弹窗内下载子用户的密码和 Access Key 信息。

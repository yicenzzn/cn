# 场景一：查看页面加载比较慢的原因

### 场景描述

某页面的页面完全加载比较慢，期望通过SGM-WEB排查原因

### 解决方案

建议接入sgm,通过页面监控进行分析

1）进入SGM-WEB页面分析模块

![](../image/Best-Practices/Best-Practices1_1.png)

2）点击图表上的“日志查询”，即可跳转至日志详情页

![](../image/Best-Practices/Best-Practices1_2.png)

3）可根据日志详情进行具体问题分析

![](../image/Best-Practices/Best-Practices1_3.png)

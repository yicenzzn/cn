# 应用场景

音视频通信（JD Real-Time Communication），是依托京东云在音视频编解码、信道传输、网络调度等核心技术，为您提供的高可用、高品质、超低延时的音视频通信服务，以及实时音视频的开发平台，您可快速搭建起多端的实时应用，满足多人音视频通话和低延时互动等场景的使用需求，JRTC 已在协同办公、远程工业检修、在线教育、金融、医疗等领域广泛使用。

## 产品功能
### 功能概述
#### 文件管理
- 支持对文件进行上传/下载/删除/分享等各类操作;
- 支持常见文件类型在线预览;
- 支持对文件进行打标，用户可以通过标签对文件进行检索;
- 支持不限上限历史版本，提供版本还原。
#### 共享协同
- 融入多项团队协作能力,支持文件在线发送、文档讨论、多人在线编辑、文件双向同步、及时消息通知等;
- 对外分享可设置密码与有效期、配置审批流，保证文件安全。
#### 企业知识沉淀
为企业打造的高效知识管理平台，能够为企业轻松搭建一个知识管理库。<br>
  - 公共文档库：用于存放企业公共资源，主要面向企业员工，展示企业公开资料。
  - 项目文档库：用于存放独立的项目资料，管理员可以通过添加项目组成员，实现项目文件多人共享、实时同步，项目组内成员在可以在其中记录内部协作产出的所有内容。

# 计费概述

京东云推出的合作弹性公网IP支持独立计费和续费，本文将介绍弹性公网IP的计费项、计费类型。


### 计费类型

合作弹性公网IP支持包年包月计费，具体信息如下表：

|计费模式|计量方式|计费项|付费周期|计费类型转换|
|--- |---|----|---|---|
|包年包月|按固定带宽计费|带宽费|预付费，购买时出账单|不支持|

> 合作弹性公网IP支持自动续费，如需长期使用资源建议配置自动续费，若未及时续费导致资源释放后将无法找回。

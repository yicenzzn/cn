# 云服务退订说明

- **按量计费后付费**：云产品直接释放/删除，不支持退款申请。

- **包年包月预付费**：如果想提前终止服务，可以根据退订规则退还资源和相应款项。

- **资源包**：如果想提前终止服务，可以根据退订规则退还资源和相应款项。

**注意事项**

1. 如果您已申请发票，申请所有类型的退订前，均需先进行退票。
2. 退订资源的关联资源，如未申请退订，可能持续产生费用。如云主机实例的关联云盘，请前往控制台确认是否需随实例释放，避免云盘持续产生费用。
3. 资源包退订后，通过资源包抵扣的产品将会切换为按量付费模式，请确保账号下资金充足。
4. 退订前请确保已完成数据迁移或备份，您可前往控制台查看实例的状态。
5. 退订申请发起后无法终止，请谨慎操作。
6. **所有产品退订申请，一个自然年内可退款总金额上限为 100,000.00 元。**

如果您在退订过程中遇到问题，可通过 [提交订单工单](https://ticket.jdcloud.com/applyorder/form?cateId=1096&questionId=1097) 或拨打400-098-8505转2处理。

## 五天无理由全额退款

对于每个主体，在一个自然年内（1月1日~12月31日），包年包月预付费云产品实例和资源包自新购订单支付成功之日起，五天内（含五天）默认享受一次无理由退款机会。

### 退款条件说明

- 为后付费产品续费，转变为预付费产品的情况，不享受无理由全额退款。

- 预付费产品如果进行过降配操作，暂不支持申请退款。

- 参与活动购买的产品，退款规则与活动规则有冲突时，以活动规则为准。活动规则中说明不支持退款的，无法申请退款。

- 推广奖励渠道订单暂不支持退款。

- 对于疑似出现异常退订/恶意退订的，京东云有权拒绝退款。

- 支持五天无理由退订的云产品及年可退实例数：

| 云产品 | 自然年内可申请无理由退订的实例个数 | 
|:---|:---|
| VPN | 1 |
| Pod | 1 |
| 专有宿主机 | 1 |
| 云主机 | 1 |
| 云硬盘 | 1 |
| 云搜索Elasticsearch | 1 |
| 云数据库RDS | 1 |
| 云数据库 MongoDB | 1 |
| 云缓存Redis | 1 |
| 云缓存Memcached | 1 |
| 原生容器 | 1 |
| 消息队列KafKa | 1 |
| 原生容器 | 1 |
| 数据传输服务DTS | 1 |
| 实例抵扣券 | 1 |
| 分析型云数据库ClickHouse | 1 |
| 分布式协调服务Zookeeper版 | 1 |
| 分布式数据库TiDB | 1 |

> 每个云产品只能全额退订一定数量的预付费实例（大部分云产品为1个）。
> 例如：一个自然年内未申请过云主机实例的五天无理由全额退款时，一笔新购订单中包含多个云主机实例，只能无理由全额退还其中1个实例的款项，其余实例不享受无理由全额退款。

针对购买时间超过五天、后付费转预付费等情况，可以根据退订规则申请 [非五天无理由部分退款](refund#context-1) 或 [续费退款](refund#context-2)。

### 退款金额说明

```退还金额 = 订单实付金额```

如果支付订单时使用了京东云发放的测试/赠送代金券，则这部分**免费代金券将不予退还**。现金/余额/电汇/付费代金券支付部分，将会全额退款。

### 返还途径说明

所有返还金额都将原路退回京东云账户。代金券支付部分退还至原代金券余额，但代金券有效期不作延期。

## 非五天无理由部分退款

不满足五天无理由全额退款规则的预付费云产品，可参考下述规则申请退订：

### 退款条件说明

- 预付费产品如果进行过降配操作，暂不支持申请退款。

- 参与活动购买的产品，退款规则与活动规则有冲突时，以活动规则为准。活动规则中说明不支持退款的，无法申请退款。

> 每个云产品只能部分退订一定数量的预付费实例（大部分云产品为10个）。
> 例如：一个自然年内已申请了9个云主机实例的非五天无理由部分退款，则再次申请部分退款时，最多只能再退订1个实例，其余实例不再享受非五天无理由部分退款。

### 退款金额说明

```退款金额 = 订单实付金额 - 资源已消耗金额```

如果支付订单时使用了京东云发放的测试/赠送代金券，则这部分**免费代金券将不予退还**。如果非免费代金券支付，且扣除已消耗金额后产品的退款金额为0，则退订仅释放资源，不会进行退款。

对于“资源已消耗金额”的计算，有以下两种方式：

**(1) 基于使用时长计算**：适用于仅有包年包月计费方式的云产品

如果申请退订日期 - 订单创建日期 > 30天，消耗金额计算公式如下：

```消耗金额 = 包年包月天单价 × 已使用时长 × 当前折扣 × 1.5```

如果申请退订日期 - 订单创建日期 <= 30天，消耗金额计算公式如下：

```消耗金额 = 包年包月天单价 × 已使用时长```

其中，订单实付金额不包括京东云发放的测试/赠送代金券支付部分，`包年包月天单价 = 订单原价 ÷ 时长`，订单原价为下单时的官网目录价。

**(2) 基于使用套餐量**：适用于资源包

```已消耗金额 = 消耗用量 × 单价 × 当前折扣```

每个产品的具体退款方式请参考对应的产品文档。

### 返还途径说明

所有返还金额都将原路退回京东云账户。代金券支付部分退还至原代金券余额，但代金券有效期不作延期。

## 续费退款

<div id="context-2"></div>

续费退款适用于未生效的续费订单，可将续费订单全额退款。续费退款后，云产品的到期时间会提前，但不会删除资源。已生效的续费订单，只能申请 [非五天无理由部分退款](refund#context-1) 。

### 退款条件说明

- 申请退款日期 早于 续费订单中的 “续费前资源到期时间”。

- 预付费产品如果进行过降配操作，暂不支持申请退款。

- 参与活动购买的产品，退款规则与活动规则有冲突时，以活动规则为准。活动规则中说明不支持退款的，无法申请退款。

- 推广奖励渠道订单暂不支持退款。

- 对于疑似出现异常退订/恶意退订的，京东云有权拒绝退款。

- 支持续费退款的云产品及年可退实例数：

| 云产品 | 自然年内可申请部分退订的实例个数 | 
|:---|:---|
| 云主机 | 100 |
| 云硬盘 | 150 |
| 云搜索Elasticsearch | 100 |
| 云缓存Redis | 100 |
| 云缓存Memcached | 100 |
| 消息队列KafKa | 100 |
| 分析型云数据库ClickHouse | 100 |
| 分布式协调服务Zookeeper版 | 100 |
| 分布式数据库TiDB | 100 |

> 每个云产品只能退订一定数量的续费实例（大部分云产品为100个）。
> 例如：一个自然年内未申请过云主机实例的续费退款时，一个未生效的续费订单中包含200台云主机，只能申请其中100台云主机的续费款项，其余实例不享受续费退款。

### 退款金额说明

```退款金额 = 续费订单实付金额```

退款金额包括所有支付方式对应的金额。

### 返还途径说明

所有返还金额都将原路退回京东云账户。代金券支付部分退还至原代金券余额，但代金券有效期不作延期。

## 降配后退款

降配后暂不支持申请退款。

## 升配后退款

包年包月云产品新购后再升配，满足五天无理由全额退款、非五天无理由部分退款或续费退款条件的，按照对应的退款规则执行。

对于非五天无理由部分退款，`消耗金额 = 新购至升配期间消耗金额 + 升配至退订期间消耗金额`
